Lecture Rating And Analysis System

A web-based feedback analysis system
developed using ASP.NET Web Forms, 
C#, and SQL Server.The project allows
students to rate lecture topics and helps
teachers analyze topic understanding through ratings.

🚀 Features
Student & Teacher Registration/Login
Student CPI and Attendance Entry
Subject & Topic Management
Topic-wise Rating System
Rating Analysis Dashboard
Automatic suggestion to reschedule topics if average rating is below 2

🛠️ Technologies Used
ASP.NET Web Forms
C#
SQL Server
HTML/CSS

📊 Modules
Student Module
Teacher Module
Rating Analysis Module

▶️ How to Run
Open the project in Visual Studio
Configure SQL Server connection string
Run the project using IIS Express
